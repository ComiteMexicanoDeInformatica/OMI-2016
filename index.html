Guía para laboratorio sede del examen

Este documento es una guía dirigida a los organizadores de la sede estatal con todos los puntos a considerar.

Computadoras

  • Para asegurar una justa competencia, todas las computadoras deben tener características similares (cpu, ram, etc.)
  • Contar con teclado en español y mouse externos, es decir, no usar trackpads en el caso de laptop.
  • Tener sistema operativo Windows 7 o más reciente.
  • Tener al menos 4GB de Memoria RAM.
  • Se deben contar con 110 computadoras para los competidores de preparatoria.
  • Se deben contar con 30 computadroas extra para los competidores de primaria.
  • Adicionalmente se necesitan 15 computadoras extras en caso de fallos o imprevistos durante el examen.

Red

  • Durante las competencias la red deberá estar restringida para que los alumnos no tengan acceso a internet.
  • Se debe agregar una excepción al firewall para que permita acceder a arena.omegaup.com:443. Todos los demás sitios deben estar bloqueados.
  • Las computadoras no deben poder ver a las demás computadoras de la red para evitar comunicacián entre los competidores.

Aplicaciones

  • Google Chrome en la versión más reciente (https://www.google.com/chrome/browser/desktop/)
  • codeblocks versión 16.01mingw (http://sourceforge.net/projects/codeblocks/files/Binaries/16.01/Windows/codeblocks-16.01mingw-setup.exe)
  • Lazarus versión 1.6 (https://sourceforge.net/projects/lazarus/files/Lazarus%20Windows%2064%20bits/Lazarus%201.6/lazarus-1.6.0-fpc-3.0.0-win64.exe/download)
  • KarelJS versión 0.2+ (https://github.com/omegaup/karel.js/releases/latest)
  • Karel versión de escritorio (http://www.olimpiadadeinformatica.org.mx/omi/omi/archivos/programas/karelomi_copy.zip)

Sesiones

  • Se tiene que crear un usuario de Windows con permisos limitados con el nombre OMI2016.

Logística

  • Se necesita que el equipo del COMI tenga acceso al laboratorio los 4 días previos al primer examen para realizar las pruebas necesarias, incluyendo sábados y domingos.
  • Durante las competencias se necesita un espacio con nodos de red (o accceso WiFi) no restringidos para el COMI.
  • Los días del examen se necesita contar con personal del laboratorio que pueda solucionar problemas de red o de equipos.
  • El primer día se realizarán simultáneamente el examen de la OMI y el de la OMIPS. Se recomienda que las áreas estén físicamente separadas para evitar distracciones.
  • Durante el examen se necesita gente que nos apoye para ayudar a los competidores a encontrar su computadora y a escoltarlos al baño para evitar que se comuniquen con otros competidores.

Si necesitan más información o tienen alguna duda, me pueden contactar escribiendo a acervantes@gestii.com.
